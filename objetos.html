1 Objetos

1.1 ¿Qué es un objeto en R?

Los objetos son valores guardados en R. Los nombres pueden ser letras, números, guiones, etc. Sin embargo, NO pueden comenzar con número.

Algunas opciones:

  1. puntos.entre.palabras
  2. guion_bajo_entre_palabras
  3. MayusculasMinusculasSeparandoPalabras

Utilizaremos la función assign para crear el objeto a

assign("a", 4)
a
## [1] 4

Pero la forma eficiente y recomendada de guardar objetos es utilizar <- (“Alt” +“-”)

a <- 4
b <- "Hola"; c <- 1:4
a
## [1] 4
b; c 
## [1] "Hola"
## [1] 1 2 3 4

1.2 Tipos de elementos

  • Integers: Números enteros
5; -6; 0
class(89L)
  • Double/ numerics: Decimales
5.3;-6.5; 0.5
class(-6.5)
  • Logical: Verdadero o Falso
TRUE; FALSE
class(TRUE)
  • Character: Cadena
"Hola Mundo";"TRUE"
## [1] "Hola Mundo"
## [1] "TRUE"
class("TRUE")
## [1] "character"
  • Valores Perdidos: Valores no disponibles. Tener cuidado al realizar operaciones con NA.
NA
## [1] NA

  • Otro tipo de valores: Existen diferentes tipos de valores que pueden ser resultado de operaciones entre objetos, entre ellos están:

  • NaN : Principalmente son el resultado de dividir cero para cero o de obtener una raiz para de un número negativo.

  • inf : Es el número que representa al infinito negativo, se obtiene dividiendo un número positivo para cero.

  • null: Es un elemento que representa el vacío se lo utiliza para borrar vectores como veremos más adeltante.

1.3 Operaciones con objetos

Las operaciones con objetos dependerán básicamente de qué tipo de objetos estemos manipulando, pero en esta ocasión veremos las operaciones más usadas.

Utilizar a R como una calculadora es bastante sencillo, podemos ir sumando(+), restando(-), dividiendo(/) y multiplicando(*), etc con objetos, esto toma de cierta manera el aspecto de poder calcular formulas con variables.

1.3.1 Ejemplo práctico

Trataremos de obtener la porción de IVA de un precio y multiplicarlo por el número de unidades. Para eso guardaremos en 2 objetos el precio inicial y el número de unidades compradas.

Precio_Inicial <-  20
N_Unidades    = 2
Precio_Final <- 20*1.12
Precio_Final
## [1] 22.4
Precio_Total <- Precio_Final*N_Unidades
Precio_Total
## [1] 44.8
